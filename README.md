# Calculator Web Application

A responsive and interactive Calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations and provides a clean, user-friendly interface for quick calculations.

## 📌 Project Overview

This calculator application allows users to perform fundamental mathematical operations such as addition, subtraction, multiplication, and division. It features a responsive design, real-time display updates, and keyboard support for an enhanced user experience.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📟 Interactive display screen
- 🗑️ Clear and reset functionality
- ⚡ Real-time input and result display
- ⌨️ Keyboard support for quick calculations
- 📱 Responsive design for different screen sizes
- 🎨 Modern and attractive user interface

## 🛠️ Technologies Used

- **HTML5** – Structure and layout
- **CSS3** – Styling and responsiveness
- **JavaScript (ES6)** – Calculator functionality and event handling
